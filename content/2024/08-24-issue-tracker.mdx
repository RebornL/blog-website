## Knit 维测日志

```
delayTask → onPreDispatch → updateData
```

- delayTask：优先加载 API 的数据
- onPreDispatch：加载组件之后，防止对应的 API 数据
- updateData：更新数据

需要补充信息：什么数据被延后加载了，什么组件被加载了，加载之后有没有待填充的数据。

## Knit 日志格式调整

- tag 只放置 `KnitOhos`
- msg 放置每个类的 tag 信息
