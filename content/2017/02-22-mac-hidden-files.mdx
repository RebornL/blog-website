## 基本命令

```bash
# 显示隐藏文件
defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

# 隐藏文件
defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder
```

## 创建快捷命令（zsh）

```bash
# 显示
echo "alias fd='defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder'" >> ~/.zshrc

# 隐藏
echo "alias fh='defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder'" >> ~/.zshrc

source ~/.zshrc
```
