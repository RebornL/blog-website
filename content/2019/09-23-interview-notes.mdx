### 回溯法 vs 分支限界法
- **回溯法**：深度优先，找出所有解
- **分支限界法**：广度优先或最小耗费优先，找一个解

## KMP 算法

```java
public static boolean kmpMatch(String str, String pattern) {
    int len1 = str.length(), len2 = pattern.length();
    if (len2 <= 0) return true;
    if (len1 <= 0) return false;
    int[] next = new int[len2];
    getNext(next, pattern);
    int i = 0, j = 0;
    while (i < len1 && j < len2) {
        if (str.charAt(i) == pattern.charAt(j)) {
            i++; j++;
        } else {
            if (j != 0) j -= (j - next[j-1]);
            else i++;
        }
    }
    return j == len2;
}
```
